| ----------- | ----------------- |
| 입력        | $0.40 / 1M tokens |
| 입력 (캐시) | $0.10 / 1M tokens |
| 출력        | $1.60 / 1M tokens |

### gpt-image-1.5 (이미지 생성, 1024×1024 기준)

| 품질   | 가격/장 |
| ------ | ------- |
| low    | $0.011  |
| medium | $0.042  |
| high   | $0.167  |

> 실제 사용: `1024×1536`, quality `medium` or `high`
> 토큰 기준 가격: 입력 $10 / 출력 $40 (1M tokens)
