> 각 시나리오는 하나의 행동만 검증합니다.

### SC-custom-001: 마법사 전 단계 완료

- **참조 규칙**: BR-custom-007
- **우선순위**: high

| 단계  | 내용                                                                                 |
| ----- | ------------------------------------------------------------------------------------ |
| Given | store 주문제작 마법사(/custom-order)에 진입한 상태                                   |
| When  | 각 단계(수량→원단→봉제→스펙→마무리→샘플→첨부)를 입력하고 최종 확인 화면까지 진행한다 |
| Then  | 단계별 정상 진입 및 최종 확인 화면이 표시된다                                        |

**경로**: `store /custom-order`
**결과**: untested
**이슈**: -
