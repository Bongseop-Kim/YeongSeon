# Custom Order 플로우

> 진입점: store /custom-order → 주문제작 마법사 (수량 → 원단 → 봉제 → 스펙 → 마무리 → 샘플 → 첨부 → 확인) → 결제

## 상태 전이 (샘플 없음)

| 현재 상태       | 다음 상태 | 방향           | 조건                   |
| --------------- | --------- | -------------- | ---------------------- |
| 대기중          | 접수      | 순방향         | 결제 완료 시 자동 전이 |
| 접수            | 제작중    | 순방향         | admin 처리             |
| 제작중          | 제작완료  | 순방향         | admin 처리             |
| 제작완료        | 배송중    | 순방향         | admin 처리             |
| 배송중          | 배송완료  | 순방향         | admin 처리             |
| 배송완료        | 완료      | 순방향         | admin 처리             |
| 결제중          | 대기중    | 결제 실패 복구 | Toss 실패 시 자동      |
| 접수            | 대기중    | 롤백           | memo 필수              |
| 제작중          | 접수      | 롤백           | memo 필수              |
| 제작완료        | 제작중    | 롤백           | memo 필수              |
| 대기중~샘플승인 | 취소      | 취소           | admin 처리             |

## 상태 전이 (샘플 있음 — fabric 기준)

| 현재 상태      | 다음 상태      | 방향   | 조건       |
| -------------- | -------------- | ------ | ---------- |
| 접수           | 샘플원단제작중 | 순방향 | admin 처리 |
| 샘플원단제작중 | 샘플원단배송중 | 순방향 | admin 처리 |
| 샘플원단배송중 | 샘플배송완료   | 순방향 | admin 처리 |
| 샘플배송완료   | 샘플승인       | 순방향 | admin 처리 |
| 샘플승인       | 제작중         | 순방향 | admin 처리 |

> sewing 샘플: 접수 → 샘플봉제제작중 → 샘플넥타이배송중 → 샘플배송완료 → 샘플승인 → 제작중
> fabric_and_sewing: 위 두 플로우 합산

## 비즈니스 규칙

- 배송중 이후(배송중, 배송완료, 완료)는 취소 불가
- 배송중 이후는 롤백 불가
- 롤백 시 memo 필수
- 샘플 단계 취소 시 sample_cost 공제 후 환불 (대기중~접수는 전액 환불)
- dimple 옵션은 tie_type=AUTO일 때만 선택 가능
- 최소 수량 없음 (견적 요청은 100개 이상)

## 고객 클레임 가능 상태

| 상태                                     | 가능한 클레임 |
| ---------------------------------------- | ------------- |
| 대기중, 결제중, 접수                     | 취소          |
| 샘플원단제작중 ~ 샘플승인 (전 샘플 상태) | 취소          |
| 배송중, 배송완료, 완료                   | 반품, 교환    |
