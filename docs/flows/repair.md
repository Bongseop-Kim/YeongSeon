# Repair 플로우

> 진입점: store /reform → 수선 아이템 추가 → 주문하기 → 결제

## 상태 전이

| 현재 상태 | 다음 상태 | 방향           | 조건                                |
| --------- | --------- | -------------- | ----------------------------------- |
| 대기중    | 결제중    | 순방향         | 결제 시작                           |
| 결제중    | 발송대기  | 순방향         | 결제 완료 시 자동 전이 **[미구현]** |
| 발송대기  | 발송중    | 순방향         | 고객 발송 처리 **[미구현]**         |
| 발송중    | 접수      | 순방향         | 판매자 수령 확인 **[미구현]**       |
| 접수      | 수선중    | 순방향         | admin 처리                          |
| 수선중    | 수선완료  | 순방향         | admin 처리                          |
| 수선완료  | 배송중    | 순방향         | admin 처리                          |
| 배송중    | 배송완료  | 순방향         | admin 처리                          |
| 배송완료  | 완료      | 순방향         | admin 처리                          |
| 결제중    | 대기중    | 결제 실패 복구 | Toss 실패 시 자동                   |
| 접수      | 대기중    | 롤백           | memo 필수                           |
| 수선중    | 접수      | 롤백           | memo 필수                           |
| 수선완료  | 수선중    | 롤백           | memo 필수                           |
| 대기중    | 취소      | 취소           | admin 처리                          |
| 발송대기  | 취소      | 취소           | admin 처리 **[미구현]**             |
| 발송중    | 취소      | 취소           | admin 처리 **[미구현]**             |

## 비즈니스 규칙

- 접수 이후(접수, 수선중, 수선완료, 배송중, 배송완료, 완료)는 취소 불가
- 배송중 이후(배송중, 배송완료, 완료, 취소)는 롤백 불가
- 롤백 시 memo 필수
- 취소 시 환불 규칙: 발송 전(대기중·발송대기)은 전액 환불, 발송 후(발송중)는 반품 택배비 공제 후 환불 [미구현]

## 고객 클레임 가능 상태

| 상태     | 가능한 클레임 | 비고                                  |
| -------- | ------------- | ------------------------------------- |
| 대기중   | 취소          | 전액 환불                             |
| 발송대기 | 취소          | 전액 환불 **[미구현]**                |
| 발송중   | 취소          | 반품 택배비 공제 후 환불 **[미구현]** |
