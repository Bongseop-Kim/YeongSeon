# Repair 플로우

> 진입점: store /reform → 수선 아이템 추가 → 주문하기 → 결제

## 상태 전이

| 현재 상태   | 다음 상태 | 방향   | 조건                   |
| ----------- | --------- | ------ | ---------------------- |
| 대기중      | 접수      | 순방향 | 결제 완료 시 자동 전이 |
| 접수        | 수선중    | 순방향 | admin 처리             |
| 수선중      | 수선완료  | 순방향 | admin 처리             |
| 수선완료    | 배송중    | 순방향 | admin 처리             |
| 배송중      | 배송완료  | 순방향 | admin 처리             |
| 배송완료    | 완료      | 순방향 | admin 처리             |
| 결제중      | 대기중    | 롤백   | memo 필수              |
| 접수        | 대기중    | 롤백   | memo 필수              |
| 수선중      | 접수      | 롤백   | memo 필수              |
| 수선완료    | 수선중    | 롤백   | memo 필수              |
| 대기중~접수 | 취소      | 취소   | admin 처리             |

## 비즈니스 규칙

- 수선중 이후(수선중, 수선완료, 배송중, 배송완료, 완료)는 취소 불가
- 배송중 이후(배송중, 배송완료, 완료, 취소)는 롤백 불가
- 롤백 시 memo 필수
- 수선 진행 전 취소 시 택배비 3,000원 제외 환불
- 수선 확인 후 취소/환불 불가

## 고객 클레임 가능 상태

| 상태   | 가능한 클레임 |
| ------ | ------------- |
| 대기중 | 취소          |
| 결제중 | 취소          |
| 접수   | 취소          |

## 화면

| 앱    | 화면      | 경로                  |
| ----- | --------- | --------------------- |
| store | 수선 주문 | /reform               |
| store | 주문 목록 | /order/order-list     |
| store | 주문 상세 | /order/:orderId       |
| admin | 주문 목록 | /orders?tab=repair    |
| admin | 주문 상세 | /orders/show/:orderId |
