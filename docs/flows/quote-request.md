# Quote Request 플로우

> 진입점: store (견적 요청 페이지) → 수량/옵션/연락처 입력 → 제출

## 상태 전이

| 현재 상태 | 다음 상태 | 방향   | 조건       |
| --------- | --------- | ------ | ---------- |
| 요청      | 견적발송  | 순방향 | admin 처리 |
| 견적발송  | 협의중    | 순방향 | admin 처리 |
| 협의중    | 확정      | 순방향 | admin 처리 |
| 확정      | 종료      | 순방향 | admin 처리 |

## 비즈니스 규칙

- 최소 수량 100개 이상
- 연락처 방법: email / kakao / phone 중 하나 필수
- 견적 금액(quoted_amount)은 admin이 견적발송 시 입력
- 고객은 견적 금액을 견적발송 이후에만 확인 가능
- 롤백 없음 (순방향 전이만 존재)

## 화면

| 앱    | 화면           | 경로                       |
| ----- | -------------- | -------------------------- |
| store | 견적 요청 목록 | /my-page/quote-request     |
| store | 견적 요청 상세 | /my-page/quote-request/:id |
| admin | 견적 요청 목록 | /quote-requests            |
| admin | 견적 요청 상세 | /quote-requests/show/:id   |
