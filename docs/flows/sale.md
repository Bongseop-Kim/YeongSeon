# Sale 플로우

> 진입점 A: 장바구니 → 주문서 → 결제 → 주문 상세
> 진입점 B: shop/:id → 주문서 (바로구매) → 결제 → 주문 상세

## 상태 전이

| 현재 상태     | 다음 상태 | 방향           | 조건                        |
| ------------- | --------- | -------------- | --------------------------- |
| 대기중        | 결제중    | 순방향         | 결제 시작                   |
| 결제중        | 진행중    | 순방향         | 토스 결제 완료 시 자동 전이 |
| 진행중        | 배송중    | 순방향         | admin 처리                  |
| 배송중        | 배송완료  | 순방향         | admin 처리                  |
| 배송완료      | 완료      | 순방향         | admin 처리                  |
| 결제중        | 대기중    | 결제 실패 복구 | Toss 실패 시 자동           |
| 진행중        | 대기중    | 롤백           | memo 필수                   |
| 대기중~진행중 | 취소      | 취소           | admin 처리                  |

## 비즈니스 규칙

- 배송중 이후(배송중, 배송완료, 완료, 취소)는 롤백 불가
- 배송중 이후(배송중, 배송완료, 완료)는 취소 불가
- 롤백 시 memo 필수
- 결제 완료 시 결제중 → 진행중 자동 전이
- 구매확정: 배송완료/배송중 상태에서 고객 직접 또는 7일 경과 시 자동 완료
