# Sale 플로우

> 진입점: 장바구니 → 주문서 → 결제 → 주문 상세

## 상태 전이

| 현재 상태     | 다음 상태 | 방향   | 조건                   |
| ------------- | --------- | ------ | ---------------------- |
| 대기중        | 진행중    | 순방향 | 결제 완료 시 자동 전이 |
| 진행중        | 배송중    | 순방향 | admin 처리             |
| 배송중        | 배송완료  | 순방향 | admin 처리             |
| 배송완료      | 완료      | 순방향 | admin 처리             |
| 결제중        | 대기중    | 롤백   | memo 필수              |
| 진행중        | 대기중    | 롤백   | memo 필수              |
| 대기중~진행중 | 취소      | 취소   | admin 처리             |

## 비즈니스 규칙

- 배송중 이후(배송중, 배송완료, 완료, 취소)는 롤백 불가
- 배송중 이후(배송중, 배송완료, 완료)는 취소 불가
- 롤백 시 memo 필수
- 결제 완료 시 결제중 → 진행중 자동 전이
- 구매확정(배송완료/완료 상태에서 고객 직접) 시 포인트 2% 적립

## 고객 클레임 가능 상태

| 상태     | 가능한 클레임 |
| -------- | ------------- |
| 대기중   | 취소          |
| 진행중   | 취소          |
| 배송중   | 반품, 교환    |
| 배송완료 | 반품, 교환    |
| 완료     | 반품, 교환    |

## 화면

| 앱    | 화면      | 경로                  |
| ----- | --------- | --------------------- |
| store | 주문 목록 | /order/order-list     |
| store | 주문 상세 | /order/:orderId       |
| admin | 주문 목록 | /orders?tab=sale      |
| admin | 주문 상세 | /orders/show/:orderId |
