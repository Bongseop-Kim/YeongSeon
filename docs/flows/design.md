# Design 플로우 (AI 디자인 생성)

> 진입점: store /design → 옵션 선택 → 생성 요청

## 플로우 단계

| 단계             | 설명                                                     | 조건                       |
| ---------------- | -------------------------------------------------------- | -------------------------- |
| 토큰 잔액 확인   | 생성 전 보유 토큰 확인                                   | 잔액 부족 시 생성 불가     |
| 옵션 선택        | AI 모델, 품질, 색상/패턴/원단 등 설정                    |                            |
| 생성 요청        | Edge Function 호출 → 토큰 차감                           | 환불 대기 중이면 사용 불가 |
| 결과 확인        | 생성된 텍스트/이미지 표시                                |                            |
| 재생성           | 동일 설정으로 재요청                                     | 토큰 추가 차감             |
| 이미지 자동 환불 | 이미지 미생성 시(데이터 부족·텍스트 전용 응답) 자동 환불 | -                          |
| 토큰 환불 신청   | paid 토큰 미사용분 수동 환불 신청 (전자거래 규정)        | pending 중복 신청 불가     |

## 비즈니스 규칙

- 토큰 차감 순서: 유료(paid) 먼저 → 보너스(bonus)
- 환불 대기(pending) 중에는 토큰 사용 불가
- text_only는 고품질(high) 미지원
- 이미지 미생성 시 자동 환불 (데이터 부족 또는 텍스트 전용 응답으로 이미지 생성이 불필요한 경우)
- paid 토큰 미사용분은 전자거래 규정에 따라 고객이 수동 환불 신청 가능 (bonus 불가)
- 동일 주문에 환불 요청 중복 신청 불가
- 신규 가입 시 bonus 토큰 30개 자동 지급

## 토큰 유형

| 유형  | 설명                            | 환불 가능 |
| ----- | ------------------------------- | --------- |
| paid  | 토큰 구매로 획득                | 가능      |
| bonus | 신규 가입 지급 또는 이벤트 지급 | 불가      |
