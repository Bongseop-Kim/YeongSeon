# 디자인 AI 토큰 정책

> 기준: 2026-03 / 환율 1,450원/USD

---

## 1. 원가 분석

### API 실비용

| 요청 유형                      | 모델                   | quality | API 원가 |
| ------------------------------ | ---------------------- | ------- | -------- |
| 텍스트만                       | gpt-4.1-mini           | —       | ~1.5원   |
| 이미지 생성 (기본)             | gpt-image-1.5          | medium  | ~62원    |
| 이미지 생성 (CI/레퍼런스 포함) | gpt-image-1.5          | high    | ~244원   |
| 텍스트만                       | gemini-2.5-flash       | —       | ~1.5원   |
| 이미지 생성                    | gemini-2.5-flash-image | —       | ~57원    |

> **문제**: 현재 OpenAI high quality(CI 이미지 포함 시)는 medium 대비 4배 비쌈.
> 현재 토큰 구조(5토큰 통일)로는 high quality 케이스에서 적자 발생 가능.

---

## 2. 토큰 소비 정책 (권장 수정안)

| 시나리오                     | 현재 토큰 | **권장 토큰** | 원가   | 판매가(30원/토큰) |
| ---------------------------- | --------- | ------------- | ------ | ----------------- |
| OpenAI 텍스트만              | 1         | 1             | ~1.5원 | 30원              |
| OpenAI 이미지 (medium)       | 5         | **5**         | ~62원  | 150원             |
| OpenAI 이미지 (high, CI포함) | 5         | **12**        | ~244원 | 360원             |
| Gemini 텍스트만              | 1         | 1             | ~1.5원 | 30원              |
| Gemini 이미지                | 3         | **3**         | ~57원  | 90원              |

**핵심 변경**: CI 이미지 또는 레퍼런스 이미지가 포함된 OpenAI 요청은 12토큰으로 차등 부과.

> `admin_settings`에서 `design_token_cost_openai_image_high` 키 추가 필요.

## 4. 무료 지급 정책

| 지급 사유 | 토큰        | 조건 |
| --------- | ----------- | ---- |
| 신규 가입 | **30 토큰** | 1회  |

> 30토큰 = Gemini 이미지 10회 또는 OpenAI medium 이미지 6회 체험 가능

---

## 5. 유효기간

| 토큰 종류      | 유효기간                   |
| -------------- | -------------------------- |
| 가입 무료 토큰 | **90일**                   |
| 구매 토큰      | **365일** (구매일 기준)    |
| 환불 토큰      | 원본 토큰의 남은 기간 승계 |

> `design_tokens` 테이블에 `expires_at` 컬럼 추가 필요.
> `get_design_token_balance`에서 만료 조건 반영 필요 (`expires_at IS NULL OR expires_at > now()`).

---

## 6. 환불 정책

| 케이스                          | 환불 여부    | 내용                       |
| ------------------------------- | ------------ | -------------------------- |
| 이미지 생성 실패 (API 오류)     | ✅ 자동 환불 | 이미지 비용 차액 즉시 환불 |
| 텍스트 분석 성공 후 이미지 실패 | ✅ 부분 환불 | 텍스트 비용(1토큰)만 차감  |
| 사용자 취소                     | ❌ 환불 없음 | 이미 API 호출된 경우       |
| 구매 토큰 현금 환불             | ❌ 불가      | 유효기간 내 미사용 포함    |
